To restore the default functionality of PSoC 5LP in Pioneer Kit (i.e., to act as Programmer for PSoC 4), bootload the PSoC 5LP with "KitProg.cyacd" using Bootloader Host application. 

KitProg.cyacd is available in <Project Install Directory>\PioneerKit_P5LP_USB_Audio\Programmer\KitProg\



For latest Kit Programmer(KitProg) firmware and information on this kit, visit:
www.cypress.com/go/CY8CKIT-042
